--PROGRAMMING LANGUAGE--

Python programming is the first language taught in BCCA. Put simply, a programming (or coding) language is a set of syntax rules that define how code should be written and formatted. Thousands of different programming languages make it possible for us to create computer software, apps and websites. During this first 3 months in BCCA, I have created and modify projects. I made a gas_pump project in python were I used a shell, core, and a disk to make the Whole Project work together as one, nevertheless I modified a GUI-Tetris game and a snake game. For the Gladiator Project I didn't use any dictionaries or special feature, however, it has the basics that make it work perfectly, for a first learner in python. In BCCA every student is allowed to have their own thoughts and ideas, therefore after the end of July students are given a set of instructions do their own creative/unique project. Snake Game

The player is represented as a snake, which grows if it eats (x). The goal of the game is to eat as many as possible without colliding into yourself. This is very easy in the early phase of the game, but is increasingly more difficult as the length of the snake grows.

This game was all written in python, We define a class Player, which holds the player's position on the screen and the speed by which it moves. In addition, we define the actions a Player instance, can do (movements).
